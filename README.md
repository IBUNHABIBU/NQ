### How to Install 

To run Library-project locally, clone the repository, navigate to it's containing directory.

#### Follow these commands step by step:-

```bash
git@github.com:IBUNHABIBU/tic-tac-toe-game.git
cd tic-tac-toe-game
```
 `$npm install `
 
 `$npm run start`
Enjoy playing it.
